# toxic-comment-classification

## Dataset

Wikipedia comments rated in 6 categories of *toxicity*.

* toxic
* severe_toxic
* obscene
* threat
* insult
* identity_hate

(dataset available on https://www.kaggle.com/c/jigsaw-toxic-comment-classification-challenge/data)
